#Manipulação de imagens

Este projeto consiste em processar imagens no formato PPM através de várias operações, nomeadamente:
-rotação;
-reflexão;
-transformação da cor(contraste e rgb);
-recorte;
-inserção de objetos(moldura e barras);
-transparência;



Para testar, deve escolher correr o programa referente à operação pretendida com redirecionamento da imagem para o output: programa.c > imagem.ppm
